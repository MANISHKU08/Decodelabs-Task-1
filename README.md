🎯 Number Guessing Game (Java)

A simple console-based Number Guessing Game developed in Java as part of the DecodeLabs Online Internship Program.

📌 Project Description

This application generates a random number between 1 and 100, and the player attempts to guess it. The program provides hints after each guess:

📉 Too Low! if the guess is smaller than the secret number.
📈 Too High! if the guess is greater than the secret number.
✅ Correct! when the player guesses the number.

The game also tracks:

Number of attempts taken
Score for each round
Total cumulative score
Option to play multiple rounds
✨ Features
Random number generation (1–100)
User-friendly console interaction
Input validation using exception handling
Score calculation based on attempts
Multiple game rounds
Total score tracking
